## 技术栈

| 层级 | 技术 | 说明 |
|------|------|------|
| **前端** | React + Vite + Tailwind | 静态网站 |
| **后端 API** | Node.js + Express | 留言、统计、搜索 |
| **数据库** | SQLite | 轻量级，无需额外服务 |
| **部署** | Nginx | 反向代理 + 静态文件 |
| **自动化** | GitHub Actions / Cron | 定时触发更新 |
| **AI** | OpenAI GPT-4o-mini | 内容处理 |
